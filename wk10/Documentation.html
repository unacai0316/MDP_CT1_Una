Project Overview

This documentation records the testing process of various sensors with ESP32, including temperature sensor (DS18B20), pressure sensor (FSR402), and tactile switch button.

Hardware Components

  • ESP32 Development Board
  • Full-size Breadboard
  • DS18B20 Temperature Sensor
  • FSR402 Pressure Sensor
  • Tactile Switch Button
  • 47KΩ Resistor
  • 220Ω Resistor
  • Various Jumper Wires

Key Findings and Lessons Learned

  • A0-A3 pins (GPIO36-39) don't support internal pullup resistors
  • Only A5 (GPIO33) showed consistent response in testing
  • Proper wire connections are crucial for sensor functionality
  • Testing each component individually helps isolate issues
