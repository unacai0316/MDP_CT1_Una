Project Overview

This project creates an interactive experience that recommends outfits based on real-world weather conditions, detected through computer vision using Teachable Machine.

Development Process & Debugging

Major Challenges

Challenge 1: ML5.js Classification Issues

Initial implementation faced problems with continuous classification causing stack overflow.

Solution:

Implemented rate limiting and proper async/await handling for classification.

Challenge 2: Image Loading and Display

Issues with loading and displaying clothing images in the correct format.

Solution:

Implemented proper preloading and error handling for images. Some naming systems issues and typos.

Challenge 3: Gesture Recognition Accuracy

Initial gesture recognition was unstable and prone to false positives.

Solution:

Added confidence threshold and improved model training.
