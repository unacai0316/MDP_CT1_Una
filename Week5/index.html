Evolution of Work

Iteration 1: Basic Setup

Initial setup of the canvas with a simple flower and basic circular motion for bees. The shapes were basic geometric forms without organic movement.

Iteration 2: Organic Shapes

Introduced the drawOrganicShape function to create more natural-looking forms for both the flower and bees. This added a slight "wobble" effect to the shapes.

Iteration 3: Enhanced Organic Movement

Improved the organic movement by increasing the range of shape deformation and adding individual noise offsets for each shape part. This created more distinct and natural-looking movements.

Iteration 4: Color Changes and Transparency

Implemented color changing for both bees and flower. bees' body change within yellowish, wings change within blueish, Flower change within warm colors.

Iteration 5: Detailed Bee Anatomy

Added more details to the bees, including eyes(white/black), a sting, and more defined wings(translate/rotate). Each part of the bee now has its own organic shape and movement, creating a dumb and cute visual. Created "drawOrganicTriangle" function to draw the sting of the bee!

Iteration 6: Fine-tuning and Performance

After I embbed the scipt into the webpage I noticed bees were moving too fast so I narrowed down the random range of "bee.speed", now the sketch won't easily make you dizzy.
